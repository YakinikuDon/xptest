## 📖 项目简介
本项目是一个基于“发疯文学”和网络亚文化语境打造的 **16型潜意识情趣倾向测试 (XPTI)**。采用了经典的极简矢量视觉风格，配以极具攻击性和戏谑感的 SBTI 风格文案。

不仅能准确剖析你在亲密关系中隐藏的“控制欲”与“受虐倾向”，还会根据最终人格匹配专属的 **「情趣 Play 推荐」** 与 **「绝佳相性猎物/主宰」**。

## 🎯 测试维度设计
我们摒弃了传统的字母拼凑，采用以下四个完全独立计算的维度矩阵：
- **倾向域 (Dom/Sub)**：掌控者 vs 服从者
- **受体域 (Mental/Physical)**：精神控制/受虐 vs 肉体控制/受虐
- **烈度域 (Vanilla/Hardcore)**：温和派 vs 硬核派
- **规则域 (Order/Chaos)**：规则崇拜 vs 无序混乱

通过 10 道直击灵魂的拷问，最终生成 16 种独一无二的深层倾向称号，如：**【暴君】**、**【教官】**、**【幻想家】**、**【苦行僧】**等。

## 🚀 特性
- 🎨 **极致的 UI 体验**：采用现代化玻璃拟物态 (Glassmorphism) 风格设计，完美适配移动端。
- 🎭 **专属人格立绘**：内置精心调教生成的 16 种独特人格精灵图，包含经典的“暗黑哲学”、“狂热信徒”等梗向造型。
- 📸 **一键长图分享**：集成了 `html2canvas` 库，用户测完后可一键生成专属雷达图与人格鉴定报告进行社交分享。
- 🔒 **纯前端架构**：完全基于 HTML/CSS/JS 开发，无后端，**绝不记录和收集任何用户个人隐私数据**。
